## HTTP
- Protocol for Data Transfer.
- uses HTTP Requests
- ### Http Requests
  - GET (to Get data from server)
  - POST (to Send data to Server)
  - PUT, HEAD, DELETE, PATCH, OPTIONS, CONNECT, TRACE (nobody uses those)
- ### HTTP Status Codes
  - Common:
    - 200 - OK
    - 404 - Not Found
    - 403 - Forbidden
  - In general : 
    - 1** - Information
    - 2** - Success
    - 3** - Redirect
    - 4** - Client Error
    - 5** - Server Error
